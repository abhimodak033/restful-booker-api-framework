# Restful Booker API Automation Framework

## Overview

This project is a reusable API automation framework developed using Python, Pytest, and Requests for testing the Restful Booker API.

## Tech Stack

* Python
* Pytest
* Requests
* JSON Schema
* Logging
* Git & GitHub

## Framework Features

* Reusable API Client
* Endpoint Layer
* Payload Management
* Pytest Fixtures
* CRUD API Automation
* Response Validation
* JSON Schema Validation
* Logging and Reporting Support

## Project Structure

```text
api_client/
config/
endpoints/
payloads/
tests/
utils/
validators/
conftest.py
pytest.ini
requirements.txt
```

## Test Scenarios

* Create Booking
* Get Booking
* Update Booking
* Partial Update Booking
* Delete Booking
* Create Token
* Health Check

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Execute tests:

```bash
pytest -v
```
